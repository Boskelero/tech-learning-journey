# GitHub Foundations Cheat Sheet

## Clone Repository

```bash
git clone REPOSITORY_URL
```

## Check Status

```bash
git status
```

## Add Files

```bash
git add .
```

## Commit Changes

```bash
git commit -m "message"
```

## Push Changes

```bash
git push origin main
```

## Pull Latest Changes

```bash
git pull origin main
```

## Create New Branch

```bash
git checkout -b feature-name
```

## Switch Branch

```bash
git checkout main
```

## Merge Branch

```bash
git merge feature-name
```

## GitHub Flow

1. Create branch
2. Make changes
3. Commit changes
4. Push branch
5. Create Pull Request
6. Review
7. Merge

## Useful GitHub Features

- Issues
- Pull Requests
- Discussions
- Projects
- Actions
- Security
- Wiki
